TorPaste is a privacy-focused pastebin service running exclusively on the Tor network. It demonstrates how web applications can be deployed as Tor hidden services using eddi - a secure application launcher that exposes web apps ONLY via Tor.

üîí How It Works

This application runs entirely over Tor using the following architecture:

  1. eddi creates a Tor v3 onion service using Arti (pure Rust Tor)
  2. Flask app runs on a Unix Domain Socket (not TCP)
  3. eddi proxies Tor connections to the Unix socket
  4. All data stays within the Tor network - no clearnet exposure
